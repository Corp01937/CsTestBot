Cloning & Setting up this Repository

Cloning in VS 2022

If you would like to test out all the commands & features that are shown in the tutorials, you can clone
this repository in your preferred IDE and run the Bot on your own system.

1. First, start by cloning the repository to your computer:

Setting up the config.json file

Next, we need to import the NuGet Packages (DSharpPlus) in this solution & we also need to input in our Token and Prefix into the "config.json" file

2. Build the Solution to import the NuGet Packages

You should get 1 Succeeded in the output, and if you check your solution in the file explorer, you should have a "../bin/Debug" Folder
which is actually the folder that contains your application when you click Start. There should also be dll files which should be the DSharpPlus packages that were imported

3. Setup the config.json file with a Token & Prefix for your Bot

Once you have imported the packages, the next step is to open the config.json file to add in your Token & Prefix
You can get a token by going into the Discord Developer Portal.

PLEASE READ: You must take this config.json file and COPY it over to the /bin/Debug folder as shown in the photo above

4. Start up the Bot

Once you have imported the NuGet packages and setup the Token/Prefix of the Bot, it is now ready to start. Click start at the top to begin execution of the Bot
You should get a message with the version of DSharpPlus you are running and your Bot should come online
